## 📝 说明

- **Source**: 使用 Source Serif Hans 字体渲染（同时也是"原始"样式的双关 😉）
- **LXGW**: 使用霞鹜文楷字体渲染
- **Flat**: 扁平化样式渲染
- **SVG**: 使用 SVG 矢量图形渲染

> 💡 提示：所有图片均使用 Napcat 平台进行渲染测试
